
## Android-PickerView

起始时间和终止时间，都已支持控制到时分秒

解决原库时间范围不控制时分秒的问题
原库地址：https://github.com/Bigkoo/Android-PickerView

#### 1.添加Gradle依赖：
```groovy
allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}

dependencies {
	        implementation 'com.github.LuckyCodeer.Android-PickerView:pickerview:4.2.8'
	}
```
