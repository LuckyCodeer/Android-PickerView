
## Android-PickerView

解决原库时间范围不控制时分秒的问题
原库地址：https://github.com/Bigkoo/Android-PickerView

#### 1.添加Gradle依赖：
```java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

dependencies {
	        implementation 'com.github.LuckyCodeer.Android-PickerView:pickerview:4.2.5'
	}
```
